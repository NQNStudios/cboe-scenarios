PLEASE NOTE: This scenario is hand tailored for a one pc party. Specifically, the one I have included. If you do not use the included party (Named "Granson") you will be drawn and quartered! No seriously, you will REALLY mess up this scenario. Plus you may make it impossible to win. If the file is corrupted, (as it often is) you can create a party yourself. I have included instructions at the bottom of this document.

Zankozzie's Big Mistake is a challenging scenario (at least I think it is) and may take you multiple tries to win. But don't worry, it isn't really that long.

There is two ways to complete Zankozzie's Big Mistake. You can finish, or win it. (Much like Redemption, an excellent scenario by Alcritas) Winning it is obviously more difficult.

The graphics are essential to this scenario. Please make sure they work. If you see a pile of rocks in the entrance to the hut where you first start, they are working. If the graphics don't work, quit and correct the problem. If you are on a mac, you will have to turn the gif (included) into a meg file. Just make a copy of a meg file you already have, change the name to "chanface.meg", and copy and paste the graphics into it.

Couldn't get the save file to work, eh? Well, here is how you can create one yourself, in only 6 easy steps!

1) Create a new party and delete all but the first pc.

2) Lower all his stats to zero and set his graphic to the funky bald guy without a shirt.*

3)Save it as "Granson".

4)Enter the Character editor, and enter the following stats: Health = 10, Strength = 2, Dexterity = 2, Intelligence = 2, Edged weapons = 3, Item lore = 1. (That's right! No Spells! Deal with it!)

5) Set gold to 3, and food to 7.

6) Delete all items and give him a pair of pants and a stone knife.

That's it!

*You could set the character to be a girl if you wanted to, but it wouldn't fit with the story quite as well.
